# ademcan.dev

## Overview
- `/` — Home page.
- `/blog` — Static blog pages.
- `/app` — All other pages.
- `components/` — All components.
- `public/` — Images for the blog and my website profile are here.
- `public/` — All posts.

## Running Locally
```bash
$ git clone https://github.com/ademcertell/website.dev.git
$ cd website
$ npm i
$ npm run dev
```
